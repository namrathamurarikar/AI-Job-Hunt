# Mode: oferta (Offer Evaluation)

Use this mode to evaluate a single job offer against the candidate profile.

## Required output
- Block A: Role summary
- Block B: CV match
- Block C: Level and strategy
- Block D: Compensation and demand
- Block E: Personalization plan
- Block F: Interview plan
- Block G: Posting legitimacy

## Scoring
- Return a final score from 1.0 to 5.0
- Include a short rationale for the score

## Persistence
- Save report in `reports/`
- Add/update tracker data in `data/applications.md`
